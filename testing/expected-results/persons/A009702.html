Download options

You can download the current document in the following formats:

WeGA

The WeGA native format as documented in our Guidelines.

TEI all

The 'standard' TEI version. WeGA element additions such as workName are mapped to the more generic name element, but no information needs to be dropped.

TEI simplePrint

"An entry-level customization, focused primarily on the needs of those encoding Western European early modern printed material" (http://www.tei-c.org/Guidelines/Customization/). This customization facilitates processing and publishing in particular, but not all WeGA concepts can be transformed. So be warned, this not a lossless transformation.

Plain text

An attempt to create a 'simple' plain text version. All variants, readings, etc. are resolved and from the text structure only paragraphs are marked by a newline. NB: This is considered an experimental feature!

More information for developers can be found in our API Dokumentation

    Biographische Informationen aus der WeGA

    • Deutscher Musikwissenschaftler mit dem Schwerpunkt europäische Musik des 17. bis 20. Jahrhunderts
    • Wurde als bislang einziger Musikwissenschaftler mit dem Gottfried-Wilhelm-Leibniz-Preis 2001 der Universität Würzburg ausgezeichnet.
    • Konrad ist ordentliches oder korrespondierendes Mitglied folgender Organisationen:
      • Bayerischen Akademie der Wissenschaften
      • Leopoldina Europaea
      • Academia Europaea
      • Akademie der Wissenschaften Göttingen
      • Akademie der Wissenschaften und der Literatur Mainz
