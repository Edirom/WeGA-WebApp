Download options

You can download the current document in the following formats:

WeGA

The WeGA native format as documented in our Guidelines.

TEI all

The 'standard' TEI version. WeGA element additions such as workName are mapped to the more generic name element, but no information had to be dropped.

TEI simplePrint

"An entry-level customization, focused primarily on the needs of those encoding Western European early modern printed material" (http://www.tei-c.org/Guidelines/Customization/). This customization facilitates processing and publishing in particular, but not all WeGA concepts can be transformed. So be warned, this not a lossless transformation.

DTA Basisformat

The DTA Basisformat is a pure TEI subset, originally used for creating the DTA Corpus. This customization is rather strict, so not all WeGA concepts can be transformed and this not a lossless transformation.

Plain text

An attempt to create a streamlined plain text version. All variants, readings, etc. are resolved and from the text structure only paragraphs are marked by a newline.

More information for developers can be found in our API Dokumentation

    Biographische Informationen aus der WeGA

    Sohn des Daguerreotypisten Carl Christian Hansen, lernte zunächst das Handwerk seines Vaters; in den 50er Jahren erlernte er die Arbeit mit Glasnegativen und Papierbildern in Deutschland; 1854 eröffnete er zusammen mit seinem Vater ein Atelier in Kopenhagen und nach zwei Jahren ein eigenes. Zahlreiche Fotografien von Hans Christian Andersen stammen von Hansen; 1864 wurde ihm der Titel als dänischer Hoffotograph verliehen; 1872 fotografierte er das in Kopenhagen wieder aufgefundene Weber-Poträt von Christian Horneman (vgl. Frank Ziegler, „… In schmucklosester Wahrheit vorgetragen“. Christian Hornemans Weber-Porträt von 1820, in: Weberiana, H. 6 (1997), S. 58–63).
