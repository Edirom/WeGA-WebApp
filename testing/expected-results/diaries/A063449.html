TEI simplePrint

TEI simplePrint ist eine TEI-Anpassung speziell für "Einsteiger", die auf die Anforderungen von westeuropäischen, gedruckten Texten der Neuzeit ausgelegt ist. Daher an dieser Stelle eine Warnung, dass dies ein verlustbehaftetes Exportformat ist.
